This directory contains an unversioned Enterprise Architect project file
of the UML model for GeoSciML v3.0 (GeoSciMLv3.0.0.eap).

It also contains the root XMI file (geosciml.xml) if you want to construct
a versioned EAP file.  Please read the instructions at 
https://www.seegrid.csiro.au/wiki/CGIModel/ConfiguringEnterpriseArchitectForGeoSciML
before you attempt this.
