    ⇑ -- dip separation sense --

    URI http://resource.geosciml.org/classifier/cgi/faultmovementtype/dip_separation_sense
    Preferred label (en) dip separation sense
    Definition () A fault along which there is some separation parallel to the dip of the fault.
    Source () GGIC vocabularies working group (Australia)
    Relationships:
    Narrower http://resource.geosciml.org/classifier/cgi/faultmovementtype/dip_slip

    URI http://resource.geosciml.org/classifier/cgi/faultmovementtype/dip_slip
    Preferred label (en) dip slip
    Definition () The net slip of the fault lies in the dip direction of the fault.
    Source () GGIC vocabularies working group (Australia)
    Relationships:
    Broader http://resource.geosciml.org/classifier/cgi/faultmovementtype/dip_separation_sense
    Notes:

    ⇑ -- extraction --

    URI http://resource.geosciml.org/classifier/cgi/faultmovementtype/extraction
    Preferred label (en) extraction
    Definition () A fault whose two sides have approached each other substantially in the direction perpendicular to the fault.
    Source () Froitzheim, N., Pleuger, J. & Nagel, T.J., 2006. Extraction faults. Journal of Structural Geology v28, pp. 1388-1395.
    Relationships:
    Narrower http://resource.geosciml.org/classifier/cgi/faultmovementtype/mixed_extraction
    Narrower http://resource.geosciml.org/classifier/cgi/faultmovementtype/pure_extraction
    Notes:

    Notes:

    ⇑ -- mixed extraction --

    URI http://resource.geosciml.org/classifier/cgi/faultmovementtype/mixed_extraction
    Preferred label (en) mixed extraction
    Definition () An extraction fault with some displacement within the fault plane.
    Source () Froitzheim, N., Pleuger, J. & Nagel, T.J., 2006. Extraction faults. Journal of Structural Geology v28, pp. 1388-1395.
    Relationships:
    Broader http://resource.geosciml.org/classifier/cgi/faultmovementtype/extraction
    Notes:

    ⇑ -- oblique slip --

    URI http://resource.geosciml.org/classifier/cgi/faultmovementtype/oblique_slip
    Preferred label (en) oblique slip
    Definition () The net slip of the fault lies between the strike and dip directions of the fault; the slip vector rakes between 10 and 80 degrees in the plane of the fault.
    Source () GGIC vocabularies working group (Australia); SMR
    Relationships:
    Narrower http://resource.geosciml.org/classifier/cgi/faultmovementtype/transpressional
    Narrower http://resource.geosciml.org/classifier/cgi/faultmovementtype/transtensional
    Notes:

    ⇑ -- pure extraction --

    URI http://resource.geosciml.org/classifier/cgi/faultmovementtype/pure_extraction
    Preferred label (en) pure extraction
    Definition () An extraction fault with no discernible displacement within the fault plane.
    Source () Froitzheim, N., Pleuger, J. & Nagel, T.J., 2006. Extraction faults. Journal of Structural Geology v28, pp. 1388-1395.
    Relationships:
    Broader http://resource.geosciml.org/classifier/cgi/faultmovementtype/extraction
    Notes:

    ⇑ -- scissor --

    URI http://resource.geosciml.org/classifier/cgi/faultmovementtype/scissor
    Preferred label (en) scissor
    Definition () A fault on which there is increasing offset or separation along the strike from an initial point of no offset, with reverse offset in the opposite direction.
    Source () Neuendorf et al 2005
    Relationships:
    Notes:

    ⇑ -- transcurrent --

    URI http://resource.geosciml.org/classifier/cgi/faultmovementtype/transcurrent
    Preferred label (en) transcurrent
    Definition () A large scale strike-slip fault in which the fault surface is steeply inclined.
    Source () GGIC vocabularies working group (Australia)
    Relationships:
    Broader http://resource.geosciml.org/classifier/cgi/faultmovementtype/strike_slip

    ⇑ -- transform --

    URI http://resource.geosciml.org/classifier/cgi/faultmovementtype/transform
    Preferred label (en) transform
    Definition () A variety of strike-slip fault along which the displacement suddenly stops or changes form; typically associated with mid-ocean ridges.
    Source () GGIC vocabularies working group (Australia)
    Relationships:
    Broader http://resource.geosciml.org/classifier/cgi/faultmovementtype/strike_slip
    Notes:

    URI http://resource.geosciml.org/classifier/cgi/faultmovementtype/transpressional
    Preferred label (en) transpressional
    Definition () A fault along which strike-slip deformation is accompanied by a component of shortening transverse to the fault.
    Source () GGIC vocabularies working group (Australia)
