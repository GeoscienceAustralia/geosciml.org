prefLabel: CGI Determination Method for Orientation Term Vocabulary
definition: This file contains the CGI Determination methods vocabulary. Methods of measuring orientation of geometrical features in rock bodies. Describes the way the dip value was determined e.g. measured, inferred from dip slope etc.

    ⇑ -- Calculated average orientation --

    URI http://resource.geosciml.org/classifier/cgi/determinationmethodorientation/calculated_average_orientation
    Preferred label (en) Calculated average orientation
    Definition () Orientation value is specified using a calculated average of a collection of related orientations (computer generalization).
    Source (en) this vocabulary
    Relationships:
    Exact match http://resource.geosciml.org/classifier/cgi/determinationmethodorientation/0001

    ⇑ -- Estimate from air photo --

    URI http://resource.geosciml.org/classifier/cgi/determinationmethodorientation/estimate_from_air_photo
    Preferred label (en) Estimate from air photo
    Definition () Orientation of a geologic structure estimated based on inspection or measurements on an air photograph.
    Source (en) this vocabulary
    Relationships:
    Exact match http://resource.geosciml.org/classifier/cgi/determinationmethodorientation/0002

    ⇑ -- Estimate from distance --

    URI http://resource.geosciml.org/classifier/cgi/determinationmethodorientation/estimate_from_distance
    Preferred label (en) Estimate from distance
    Definition () Orientation of a geologic structure based on observation from a distance great enough to preclude direct inspection of the structure to determine orientation.
    Source (en) this vocabulary
    Relationships:
    Exact match http://resource.geosciml.org/classifier/cgi/determinationmethodorientation/0003

    ⇑ -- Measure on outcrop --

    URI http://resource.geosciml.org/classifier/cgi/determinationmethodorientation/measure_on_outcrop
    Preferred label (en) Measure on outcrop
    Definition () Orientation of surface or line is measured on an outcrop of that surface or line directly, e.g. by measuring a particular bedding surface, a 3-D exposure of a fold hinge, a particular stretched mineral grain in a foliation surface.
    Source (en) this vocabulary
    Relationships:
    Exact match http://resource.geosciml.org/classifier/cgi/determinationmethodorientation/0004
    Narrower Standard on site measure
    Narrower Visual surface estimation on outcrop

    ⇑ -- Method unknown --

    URI http://resource.geosciml.org/classifier/cgi/determinationmethodorientation/method_unknown
    Preferred label (en) Method unknown
    Definition () use value qualifier to specify the kind of nil
    Source (en) this vocabulary
    Relationships:
    Exact match http://resource.geosciml.org/classifier/cgi/determinationmethodorientation/0005

    ⇑ -- Photogeologic determination --

    URI http://resource.geosciml.org/classifier/cgi/determinationmethodorientation/photogeologic_determination
    Preferred label (en) Photogeologic determination
    Definition () Orientation determined based on measurements from aerial photography or satellite imagery (in conjunction with an elevation model.
    Source (en) this vocabulary
    Relationships:
    Exact match http://resource.geosciml.org/classifier/cgi/determinationmethodorientation/0006

    ⇑ -- Standard on site measure --

    URI http://resource.geosciml.org/classifier/cgi/determinationmethodorientation/standard_on_site_measure
    Preferred label (en) Standard on site measure
    Definition () Orientation measured using compass or other instrument directly on or at an outcrop of the structure.
    Source (en) this vocabulary
    Relationships:
    Exact match http://resource.geosciml.org/classifier/cgi/determinationmethodorientation/0007
    Broader Measure on outcrop

    ⇑ -- Three point determination --

    URI http://resource.geosciml.org/classifier/cgi/determinationmethodorientation/three_point_determination
    Preferred label (en) Three point determination
    Definition () Orientation determined by fitting a plane to three or more points located on the geologic surface of interest.
    Source (en) this vocabulary
    Relationships:
    Exact match http://resource.geosciml.org/classifier/cgi/determinationmethodorientation/0008

    ⇑ -- Visual surface estimation on outcrop --

    URI http://resource.geosciml.org/classifier/cgi/determinationmethodorientation/visual_surface_estimation_on_outcrop
    Preferred label (en) Visual surface estimation on outcrop
    Definition () Orientation of a surface is measured by visually averaging across one or more outcrops in a small area--e.g. approximating dip by looking down strike of beds, approximating strike by outcrop trace of one or more beds.
    Source (en) this vocabulary
    Relationships:
    Exact match http://resource.geosciml.org/classifier/cgi/determinationmethodorientation/0009
    Broader Measure on outcrop
