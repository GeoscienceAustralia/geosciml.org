prefLabel: CGI Value qualifier vocabulary
definition: This file contains the CGI Value Qualifier code vocabulary. These terms are taken from the enumeration for values that may be assigned as the valueQualifier on CGI_Value. Definitions are proposed. This codelist includes qualifiers of several sorts: 1. frequence of occurrence--qualifiers that indicate how many occurrences of a described feature are expected to have a reported value. E.g. 'commonly red, sometimes brown'; 2. kinds of statistical procedure used to determine a single value to report based on a collection of observed values, e.g. mean, median, mode.; 3. kinds of nil values; 4. kind of value--a limiting value or the actual estimated value.
