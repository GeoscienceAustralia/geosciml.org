    ⇑ -- dip separation sense --

    URI http://resource.geosciml.org/classifier/cgi/faultmovementtype/0001
    Preferred label (en) dip separation sense
    Definition () A fault along which there is some separation parallel to the dip of the fault.
    Source () GGIC vocabularies working group (Australia)
    Relationships:
    Narrower http://resource.geosciml.org/classifier/cgi/faultmovementtype/0002
    Notes:

    ⇑ -- dip slip --

    URI http://resource.geosciml.org/classifier/cgi/faultmovementtype/0002
    Preferred label (en) dip slip
    Definition () The net slip of the fault lies in the dip direction of the fault.
    Source () GGIC vocabularies working group (Australia)
    Relationships:
    Broader http://resource.geosciml.org/classifier/cgi/faultmovementtype/0001
    Notes:

    ⇑ -- extraction --

    URI http://resource.geosciml.org/classifier/cgi/faultmovementtype/0003
    Preferred label (en) extraction
    Definition () A fault whose two sides have approached each other substantially in the direction perpendicular to the fault.
    Source () Froitzheim, N., Pleuger, J. & Nagel, T.J., 2006. Extraction faults. Journal of Structural Geology v28, pp. 1388-1395.
    Relationships:
    Narrower http://resource.geosciml.org/classifier/cgi/faultmovementtype/0005
    Narrower http://resource.geosciml.org/classifier/cgi/faultmovementtype/0007
    Notes:

    URI http://resource.geosciml.org/classifier/cgi/faultmovementtype/0004
    Preferred label (en) horizontal
    Definition () The fault is horizontal.
    Source () GGIC vocabularies working group (Australia)
    Relationships:
    Notes:

    ⇑ -- mixed extraction --

    URI http://resource.geosciml.org/classifier/cgi/faultmovementtype/0005
    Preferred label (en) mixed extraction
    Definition () An extraction fault with some displacement within the fault plane.
    Source () Froitzheim, N., Pleuger, J. & Nagel, T.J., 2006. Extraction faults. Journal of Structural Geology v28, pp. 1388-1395.
    Relationships:
    Broader http://resource.geosciml.org/classifier/cgi/faultmovementtype/0003
    Notes:

    ⇑ -- oblique slip --

    URI http://resource.geosciml.org/classifier/cgi/faultmovementtype/0006
    Preferred label (en) oblique slip
    Definition () The net slip of the fault lies between the strike and dip directions of the fault; the slip vector rakes between 10 and 80 degrees in the plane of the fault.
    Source () GGIC vocabularies working group (Australia); SMR
    Relationships:
    Narrower http://resource.geosciml.org/classifier/cgi/faultmovementtype/0013
    Narrower http://resource.geosciml.org/classifier/cgi/faultmovementtype/0014
    Notes:

    ⇑ -- pure extraction --

    URI http://resource.geosciml.org/classifier/cgi/faultmovementtype/0007
    Preferred label (en) pure extraction
    Definition () An extraction fault with no discernible displacement within the fault plane.
    Source () Froitzheim, N., Pleuger, J. & Nagel, T.J., 2006. Extraction faults. Journal of Structural Geology v28, pp. 1388-1395.
    Relationships:
    Broader http://resource.geosciml.org/classifier/cgi/faultmovementtype/0003
    Notes:

    ⇑ -- scissor --

    URI http://resource.geosciml.org/classifier/cgi/faultmovementtype/0008
    Preferred label (en) scissor
    Definition () A fault on which there is increasing offset or separation along the strike from an initial point of no offset, with reverse offset in the opposite direction.
    Source () Neuendorf et al 2005
    Relationships:
    Notes:

    URI http://resource.geosciml.org/classifier/cgi/faultmovementtype/0010

    ⇑ -- transcurrent --

    URI http://resource.geosciml.org/classifier/cgi/faultmovementtype/0011
    Preferred label (en) transcurrent
    Definition () A large scale strike-slip fault in which the fault surface is steeply inclined.
    Source () GGIC vocabularies working group (Australia)
    Relationships:
    Broader http://resource.geosciml.org/classifier/cgi/faultmovementtype/0010

    ⇑ -- transform --

    URI http://resource.geosciml.org/classifier/cgi/faultmovementtype/0012
    Preferred label (en) transform
    Definition () A variety of strike-slip fault along which the displacement suddenly stops or changes form; typically associated with mid-ocean ridges.
    Source () GGIC vocabularies working group (Australia)
    Relationships:
    Broader http://resource.geosciml.org/classifier/cgi/faultmovementtype/0010
    Notes:

    ⇑ -- transpressional --

    URI http://resource.geosciml.org/classifier/cgi/faultmovementtype/0013
    Preferred label (en) transpressional
    Definition () A fault along which strike-slip deformation is accompanied by a component of shortening transverse to the fault.
    Source () GGIC vocabularies working group (Australia)
    Relationships:

    Source () GGIC vocabularies working group (Australia)
    Relationships:
    Broader http://resource.geosciml.org/classifier/cgi/faultmovementtype/0006

    URI http://resource.geosciml.org/classifier/cgi/faultmovementtype/0015
