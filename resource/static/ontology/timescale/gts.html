Introduction

This is an RDF/OWL representation of the GeoSciML Geologic Timescale model, which has been adapted from the model described in Cox, S.J.D, & Richard, S.M. (2005) A formal model for the geologic timescale and GSSP, compatible with geospatial information transfer standards, Geosphere, Geological Society of America 1/3, 119–137.

Hierarchical Time Ordinal Reference Systemc back to ToC or Class ToC

IRI: http://resource.geosciml.org/ontology/timescale/thors#ReferenceSystem

is defined by
http://resource.geosciml.org/ontology/timescale/thors

A temporal ordinal reference system is a constrained temporal topological complex, in which the edges are called 'eras'. Each era may be subdivided into one set of eras of the next finer rank. This constraint is required to establish a mono-hierarchy, so that a set of temporal positions defined relative to the reference system may be uniquely ordered.

has super-classes
ordinal reference system
topological complex
concept scheme
reference point in timescaleop min 2
temporal reference system componentop min 1
has sub-classes
Geologic Timescalec
is in domain of

is in range of
systemop
