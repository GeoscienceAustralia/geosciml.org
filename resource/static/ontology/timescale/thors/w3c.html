Abstract

This graph provides an alignment of the THORS ontology with OWL-Time, as an alternative to the use of ISO 19108 classes as the basis for the ontology. Where appropriate, THORS classes and properties are related to OWL-Time classes and properties using rdfs:subClassOf, rdfs:subPropertyOf and owl:equivalentProperty axioms. Where the latter apply, the THORS properties are also annotated "deprecated" with the expectation that the OWL-Time properties will be used instead.
