GM_Clothoid (Figure 19) implements the clothoid (or Cornu's spiral), which is a plane curve whose curvature is a fixed function of its length. <br/>This geometry is mainly used as a transition curve between curves of type straight line/circular arc or circular arc/circular arc.<br/>
