<font color="#4f81bd">GeologicSpecimenView</font> is a simplified view of a point-located specimen from GeoSciML <font color="#4f81bd">GeologicSpecimen</font> (an extension of Observations & Measurements - ISO19156) with key property values summarised as labels (unconstrained character strings) or arbitrarily selected classifiers to be used for thematic mapping purposes. The latter are the properties suffixed with “_uri” and will contain URIs referring to controlled concepts in published vocabularies.<br/><br/>

Notes: Globally unique identifier (eg, an IGSN sample number).  Should have the same value as a corresponding GeoSciML GeologicSpecimen.  Globally unique <font color="#4f81bd">identifier:Primitive::CharacterString</font> shall uniquely identifies a tuple within the dataset. Identifiers shall be formatted as URI according to RFC 3986.  This URI could be used to access more detailed, such as a GeoSciML Basic, representation of the feature<br/><br/>If present, the URI should resolve to a representation that corresponds to an instance of GeoSciML GeologicSpecimen.<br/><br/>
