A reference specimen is a specimen with known or accepted values of some property.  The citation property describes the location of a published description of these values. Reference specimens include analytical blanks. <font color="#4f81bd">ReferenceSpecimens</font> are used in quality control procedures to assess method reproducibility. <br/><br/>Analytical results from a reference specimen analysed during an <font color="#4f81bd">AnalyticalSession</font> are delivered in the same way as the results of other specimens analysed in that session.<br/><br/><br/>
