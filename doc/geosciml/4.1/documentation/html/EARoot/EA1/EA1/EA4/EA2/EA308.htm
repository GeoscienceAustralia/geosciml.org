<font color="#4f81bd">PhysicalDescription</font> is a class that describes the numeric physical properties of a geologic unit (<font color="#4f81bd">GeologicUnit</font>) earth material (<font color="#4f81bd">EarthMaterial</font> ), or geologic structure (<font color="#4f81bd">GeologicStructure</font>) (e.g., density, porosity, magnetic susceptibility, remanent magnetism).  These properties are modelled here as scalar numeric values (<font color="#4f81bd">SWE::Quantity</font>).<br/><br/>
