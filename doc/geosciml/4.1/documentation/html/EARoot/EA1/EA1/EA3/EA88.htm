GeologyBasic is a package of classes representing fundamental geological map features and the relations between them.  GeoSciML describes a geologic dataset as a series of <font color="#4f81bd">GeologicFeature</font> occurrences, spatially represented as <font color="#4f81bd">MappedFeature</font>.  A map is a collection of <font color="#4f81bd">MappedFeatures</font>.  The term “map”, typically understood as a map sheet (a given area on the surface of the earth), is only one possible collection of <font color="#4f81bd">MappedFeatures</font>.  Other examples are cross-sections, block diagrams, and even borehole logs (a linear map).  <font color="#4f81bd">MappedFeature</font> can represent any features and <font color="#4f81bd">GeologicFeatures</font> are one of the kinds of features it can represent.  A <font color="#4f81bd">MappedFeature</font> identifies what it represents using its “specification” association.  <br/><br/>
