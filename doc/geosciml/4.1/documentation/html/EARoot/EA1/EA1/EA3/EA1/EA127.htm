A <font color="#4f81bd">CompoundMaterial</font> is an <font color="#4f81bd">EarthMaterial</font> composed of particles made of <font color="#4f81bd">EarthMaterials</font>, possibly including other <font color="#4f81bd">CompoundMaterials</font>.  This class is provided primarily as an extensibility point for related domain models that wish to import and build on GeoSciML, and wish to define material types that are compound but are not rock or rock-like material. In the context of GeoSciML "<font color="#4f81bd">RockMaterial</font>" should be used to describe units made of rock.  <br/><br/>
