<font color="#4f81bd">GSML_GeometricDescriptionValue</font> is a special abstract data type for descriptions of planar or linear orientations of a geologic feature.   Different subtypes allow specifying direction by direction vector (e.g. dip/dip direction), compass point (e.g.  NE), or description (e.g. "toward fold hinge", "below').<br/><br/>
