A package of classes to describe the planar or linear orientation of a geologic feature. Allows specifying direction by a numerical direction vector (eg; dip/dip direction), or a description (eg; compass point (NE), or other text - "toward fold hinge", "below").<br /></p><p><br /></p><p>An additional GSML_QuantityRange class extends sweCommon:QuantityRange to allow upper and lower values in a numerical range to be delivered as two separate attributes. This is to facilitate FES query of upper and lower values.<br /></p>
