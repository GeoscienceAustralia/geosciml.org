Recommended profile for delivering geochemical analysis results.<br /></p><p><br /></p><p>1. One element (or analyte) analysis result per observation<br /></p><p>2. Use swe:Quantity to deliver measurement values in "result:any"<br /></p><p>3. Use swe:uom to deliver an HTTP-URI to the UCUM register.  Use the "code" attribute to deliver a text label for the unit of measure.<br /></p><p>4. Use swe:Quantity/definition to deliver an HTTP-URI to a vocabulary term to describe the quantity being analysed.<br /></p><p>5. Use swe:label to deliver a human readable "column heading"<br /></p><p>6.
