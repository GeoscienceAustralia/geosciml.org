The GeoSciML GeologicUnit Package contains classes representing notional geologic units, whose complete and precise extent is inferred to exist. Explicit spatial properties are available through association with a MappedFeature. <br /></p><p><br /></p><p>Geologic units includes both formal units (i.e. formally adopted and named in the official lexicon) and informal units (i.e. named but not promoted to the lexicon) and unnamed units (i.e. recognisable and described and delineable in the field but not otherwise formalised).<br /></p><p><br /></p><p>Change in version 3.2 --> The GeologicUnit package now imports CGI_Utilities in order to gain access to the new GSML_QuantityRange element (an extension of swe:QuantityRange).<br /></p>
