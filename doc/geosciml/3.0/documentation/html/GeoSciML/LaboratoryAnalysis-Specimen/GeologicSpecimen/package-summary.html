Package GeoSciML/LaboratoryAnalysis-Specimen/GeologicSpecimen

The GeologicSpecimen package extends the ISO19156 O&amp;M schema, and describes processes relevant to the sampling, preparation and analysis of geologic specimens.

ReferenceSpecimen
<<FeatureType>> 		A reference specimen is a specimen with known or accepted values of some property. The citation property describes the location of a published description of these values. Reference specimens include analytical blanks.

Analytical results from a reference specimen analysed during an AnalyticalSession are delivered in the same way as the results of other specimens analysed in that session.
