feature attribute<br /></p><p> characteristic of a feature<br /></p><p><br /></p><p>NOTE: A feature attribute has a name, a data type, and a value domain associated to it.  A feature attribute for a feature instance also has an attribute value taken from the value domain.  <br /></p><p>EXAMPLE: A feature attribute named 'color' may have an attribute value 'green' which belongs to the data type 'text'.  A feature attribute named 'length' may have an attribute value '82.4' which belongs to the data type 'real'.<br /></p><p>[Victoria Terminology Harmonization Meeting]<br /></p>
