This directory contains an unversioned Enterprise Architect project file of the UML model for GeoSciML v3.1 (GeoSciMLv3.1_unversioned.eap). 

If you wish to construct an EAP file using UML files in the version-controlled GeoSciML Subversion Repository, please follow the instructions at https://www.seegrid.csiro.au/wiki/CGIModel/ConfiguringEnterpriseArchitectForGeoSciML and load the GeoSciML model from here: https://www.seegrid.csiro.au/subversion/GeoSciML/tags/3.1.
