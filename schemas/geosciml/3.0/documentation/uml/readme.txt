This directory contains an unversioned Enterprise Architect project file of the UML model for GeoSciML v3.0 (GeoSciMLv3.0.0.eap).

If you wish to construct an EAP file using UML files in the version-controlled GeoSciML Subversion Repository, please follow the instructions at https://www.seegrid.csiro.au/wiki/CGIModel/ConfiguringEnterpriseArchitectForGeoSciML and load the GeoSciML model from here: https://www.seegrid.csiro.au/subversion/GeoSciML/tags/3.0.0.
